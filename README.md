# React Cards Game

Juego de cartas por turnos inspirado en Axie Infinity. Construye tu equipo de criaturas, enfréntate a enemigos en el mapa y vence a los bosses para ganar.

## Características

- **8 clases de criaturas**: Planta, Bestia, Acuático, Ave, Insecto, Reptil, Amanecer y Crepúsculo
- **Sistema de batalla por turnos**: Usa cartas con diferentes efectos (daño, escudo, curación, estados alterados)
- **Estados alterados**: Sangrado, Veneno, Aturdimiento, Congelamiento, buffs de ATK/DEF
- **3 mapas**: Pradera Esmeralda, Costa Tempestuosa, Cavernas Malditas
- **Jefes**: Enfrenta a bosses únicos en cada mapa

## Requisitos

- Node.js 18+
- npm

## Instalación

```bash
npm install
```

## Ejecutar

```bash
npm run dev
```

Abre http://localhost:5173 en tu navegador.

## Como jugar

1. **Título**: Presiona cualquier botón para comenzar
2. **Selección**: Elige 3 criaturas de las clases disponibles
3. **Mapa**: Selecciona un mapa y avanza por las etapas
4. **Batalla**: 
   - Tienes 3 cartas por turno
   - Cada carta cuesta energía (1-2)
   - Gana quien derribe al equipo enemigo

## Construir para producción

```bash
npm run build
```
